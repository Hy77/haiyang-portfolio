Active RGB-D Camera Positioning for Robust Object Detection

A RealSense D435-based active perception system that dynamically optimizes camera viewpoints to improve object detection confidence under occlusion, distance, and lighting challenges.

Problem

Detect a target object reliably in cluttered scenes with occlusion, poor viewpoint, and strong light interference.

My Role

Designed the detection logic, confidence improvement strategy, object finding strategy, and real-time camera guidance workflow.

Outcome

Built a real-time RGB-D perception system that guided viewpoint adjustments to improve detection reliability and recover hidden targets.

System Overview

This project explored active perception using an Intel RealSense D435 camera. The system first detects a coke can using HSV colour filtering and depth information, then evaluates whether the current viewpoint is suitable. If confidence is low, it issues camera movement instructions to improve alignment, reduce occlusion, and avoid glare. If the can cannot be detected, the system initiates an object-finding strategy to explore less cluttered regions and search behind obstacles.

Detection Module

The detection pipeline combines HSV colour filtering with depth-based dimension checks to identify a coke can candidate. The colour filter isolates likely red targets, while depth data from the RealSense D435 is used to estimate object size and validate whether the detected object matches expected coke can dimensions.

Confidence Improvement Strategy

When detection confidence drops below the desired threshold, the system uses a reward-based strategy to improve visibility. Adjustments are prioritised based on whether the object is off-centre, too far, too close, partially blocked, or affected by strong light reflections. The result is a practical, real-time guidance system for improving the camera viewpoint.

Object Finding Strategy

If the target object is not detected at all, the system switches to an exploration mode. It segments visible objects on the left and right sides of the frame, identifies less cluttered directions, and recommends motion paths that help the camera search behind obstacles and recover the target.

Quantitative Results

Experimental results showed that the system increased detection confidence over time as the camera navigated around obstacles and refined its viewpoint. Additional metrics tracked distance to the target and the number of bright light spots, demonstrating that the system could both approach the object effectively and reduce glare through repositioning.

Future Direction

While the current implementation provides movement guidance to a human operator, the system was designed as a foundation for future fully autonomous camera control. The project demonstrates how active viewpoint optimisation can improve object detection reliability and provides a strong basis for future robotic perception and autonomous inspection systems.
